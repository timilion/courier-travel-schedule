# courier-travel-schedule


Инструкция
------------
- мин. версия php - 7.4
- Изменить конфигурацию БД  - /config/db.php
- Запустить создание и заполнение таблиц - $ php scripts/createTable.php 
 
 
Затем вы можете получить доступ к приложению по следующему URL-адресу:
~~~
http://localhost/web/
~~~
